# RPA para Consulta de Tributações de NCM no Econet

Este projeto é um **Robotic Process Automation (RPA)** desenvolvido em Python que utiliza a biblioteca **Selenium** para realizar consultas automáticas das tributações de códigos NCM em uma planilha e preencher os resultados de forma automatizada.

## 📋 Funcionalidades

- **Leitura de planilha**: Lê uma lista de códigos NCM de uma planilha Excel.
- **Automação de consulta**: Realiza consultas no site da Econet para buscar as tributações de cada NCM.
- **Preenchimento automático**: Atualiza os resultados das tributações na planilha original.
- **Relatório de erros**: Registra códigos NCM com falhas na consulta.

---

## 🛠️ Tecnologias Utilizadas

- **Python**: Linguagem principal para desenvolvimento.
- **Selenium**: Para automação de navegação no navegador.
- **Pandas**: Para manipulação e leitura/escrita de planilhas.
- **OpenPyXL**: Para integração com o Excel.
- **Google Chrome e ChromeDriver**: Navegador e driver necessários para o Selenium.
